### docsify步骤

初始化目录

```
docsify init ./docs
```

使用sidebar，并且自定义名字

```
<script>
  window.$docsify = {
    loadSidebar: 'sidebar.md'
  }
</script>
```

预览页面

```
docsify serve docs
```
