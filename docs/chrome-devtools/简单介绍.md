网址： https://developers.google.com/web/tools/chrome-devtools

Chrome开发者工具是一套内置于Google Chrome中的Web开发和调试工具，可以用来帮助你快速编写页面，定位各种问题，最终帮你更快更好的构建网站。

打开开发者工具:

- 页面上右键，选择“检查”跳转到元素面板
- 快捷键：Command+Option+J (Mac) or Control+Shift+J (Windows, Linux, Chrome OS) ，跳转到控制台面板
- ...

这边基于85版本的chrome浏览器，介绍下开发者工具里的部分面板

介绍：

- 检查和调整页面
- 检查和调整页面样式
- 调试JavaScript
- 在控制台面板上查看信息，与javascript交互
- 优化网站速度
- 检查网络活动
