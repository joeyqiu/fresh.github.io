## The Basics

The goal of TypeScript is to be a static type checker for Javascript programs - in other words, a tool that runs before your code runs(static) and ensures that the types of the program are correct(typechecked).

TS的目标就是作为JS预发的静态类型检查器 - 作为一个工具，在你的代码运行之前先静态跑一遍，并通过类型检查来确保程序中的类型都是正确的。
