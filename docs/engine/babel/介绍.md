Babel是一个工具链，主要用来把最新的ES语法转换成向后兼容的语法，是的在旧版本的浏览器中能够运行，下面是Babel主要做的事情：

- Transform Syntax (语法转换)
- Polyfill features that are missing in your target environment(through @babel/polyfill)
- Source code transformation(codemods)
- And more...



Babel is a compiler (source code => output code). Like many other compilers it runs in 3 stages: parsing, transforming, and printing.  解析、转换(插件作用的阶段)、生成。

AST：abstract syntax tree，抽象语法树。

Babel只做语法转换相关的事情，把require等代码集合起来是webpack的事情。
