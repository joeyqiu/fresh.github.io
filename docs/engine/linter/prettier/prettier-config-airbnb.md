## prettier-config-airbnb

https://www.npmjs.com/package/prettier-config-airbnb

A [Prettier](https://prettier.io/) configuration based on the [Airbnb JavaScript style guide](https://github.com/airbnb/javascript)



prettier-config-airbnb的最终配置，其实是下面这些。

```json
{
  "printWidth": 100,
  "tabWidth": 2,
  "useTabs": false,
  "semi": true,
  "singleQuote": true,
  "quoteProps": "as-needed",
  "jsxSingleQuote": false,
  "trailingComma": "all",
  "bracketSpacing": true,
  "jsxBracketSameLine": false,
  "arrowParens": "always"
}

```
