## Available Scripts

create-react-app中提供了四个命令



### npm start

或者是`yarn start`

以开发者模式启动app，并在浏览器中打开`http://locahost:3000`



### npm test

或者是`yarn test`

启动测试，没用过。。。。



### npm run build

或者是`yarn build`

以生产者模式打包应用，并放到`build`目录下去。

It correctly bundles React in production mode and optimize the build for the best performance.

The build is minified and the filenames include the hashes, You app is ready to be deployed.



### npm run eject

或者是`yarn eject`

这个操作是不可后退的，需要谨慎使用。

如果你对create-react-app提供的打包工具不是很满意，需要自己去做定制的话，就需要使用`eject`命令了。改命令把webpack等配置文件抽离到项目目录下的`config`目录，可以让你自由配置。

如果你直接在`node_modules/react-script`里面修改配置的话，启动脚本有时候会拉去最新代码，会把你的配置覆盖掉，所以需要自定义配置，就要用`eject`命令了。
